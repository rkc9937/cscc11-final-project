# CSCC11 Final Project
## Instructions?
1. First, download the [MCI dataset](https://drive.google.com/drive/folders/1dtYl4JeOrJ33C3hirnAupLWKlnGeW4dj?usp=sharing).
2. Place the downloaded mci.csv within the data folder.
3. Run main.py: ensure that you have the proper libraries installed in your virtual environment.
4. Now you can run any model, by `python model_name.py`

>[!NOTE]
> After step 3, you should see `mci_formatted.pkl` within your data file.
